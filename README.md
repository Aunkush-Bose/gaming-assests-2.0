# 🎮 Gaming Assets Smart Contract

### Built on Soroban (Stellar Smart Contracts)

---

## 📌 Project Description

The **Gaming Assets Smart Contract** is a decentralized system built using **Soroban**, enabling players to securely own, manage, and transfer in-game assets on the **Stellar blockchain**.

This project introduces a **trustless digital inventory system**, where assets are no longer controlled by centralized game servers but are instead owned and verifiable on-chain.

## 🚀 What it does

* 🪙 Mint unique gaming assets tied to player accounts
* 🔄 Transfer assets securely between players
* 📦 Store asset metadata directly on-chain
* 🔍 Retrieve asset information instantly

## ✨ Features

* 🔐 **Secure Ownership** — authenticated via Stellar addresses
* 🎮 **Decentralized Inventory** — each player has on-chain assets

## 📂 Smart Contract Functions

### 🪙 `mint_asset(player, asset_id, metadata)`

Creates and assigns a new asset to a player.

### 🔁 `transfer_asset(from, to, asset_id)`

Transfers ownership of an asset from one player to another.

### 🔍 `get_asset(player, asset_id)`

Fetches metadata of a specific asset.
